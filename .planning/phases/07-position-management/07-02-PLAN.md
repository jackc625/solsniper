**Recursive setTimeout pattern (critical — NOT setInterval):**
```typescript
private scheduleTick(): void {
  this.timer = setTimeout(async () => {
    try {
      await this.tick();
    } catch (err) {
      log.error({ err }, 'PositionManager tick threw unexpectedly');
    } finally {
      if (this.running) this.scheduleTick();
    }
  }, this.config.positionManagement.pollIntervalMs);
}
```

**tick() method:**
```typescript
private async tick(): Promise<void> {
  const positions = this.tradeStore.getMonitoringTrades();
  await Promise.allSettled(positions.map(p => this.evaluatePosition(p)));
}
```

**getPositionValueSol() helper — exact pattern from research:**
```typescript
private async getPositionValueSol(mint: string, tokenAmountRaw: bigint): Promise<number | null> {
  const SOL_MINT = 'So11111111111111111111111111111111111111112';
  const url = `https://api.jup.ag/swap/v1/quote?inputMint=${mint}&outputMint=${SOL_MINT}` +
    `&amount=${tokenAmountRaw.toString()}&slippageBps=50&maxAccounts=64`;
  try {
    const resp = await fetch(url);
    if (!resp.ok) return null;
    const data = await resp.json() as { outAmount: string };
    return Number(data.outAmount) / 1e9;
  } catch {
    return null;
  }
}
```

**evaluatePosition() logic (per locked decisions):**
1. If `sellsInFlight.has(trade.mint)` → return (double-sell guard)
2. PumpPortal backfill if amountTokens is nullish (see above)
3. Compute `tokenAmountRaw = BigInt(Math.round(amountTokens))` — NEVER `BigInt(amountTokens)` directly (pitfall 2)
4. `currentValueSol = await getPositionValueSol(mint, tokenAmountRaw)` — if null, return
5. Guard: if `trade.amountSol == null` return (no entry price to compare against)
6. `ratio = currentValueSol / trade.amountSol`
7. Update high watermark: `newWatermark = Math.max(watermarks.get(mint) ?? trade.amountSol, currentValueSol)` — initialized to entry price per locked decision
8. **Exit evaluation (TP takes priority over SL per locked decision):**
   - Check tiered TP first: if `activeTier && ratio >= activeTier.at` → tiered TP sell
   - Else check trailing stop (if `trailingStopPct > 0`): if `currentValueSol < newWatermark * (1 - trailingStopPct/100)` → trailing stop sell
   - Else check SL: if `ratio < (1 + stopLossPct/100)` → stop-loss sell

9. **On sell trigger:**
   - `sellsInFlight.add(mint)`
   - `void this.sellLadder.sell(mint, tokensToSell)` — fire and forget, SellLadder handles MONITORING→SELLING transition internally
   - For tiered TP: `tierIndices.set(mint, nextTierIndex + 1)` to advance to next tier
   - Chain cleanup: after the sell promise settles, `sellsInFlight.delete(mint)` — use `.finally()` on the returned promise (note: `void` discards the promise for the tick, but capture the promise first: `const p = this.sellLadder.sell(mint, tokensToSell); void p; p.finally(() => this.sellsInFlight.delete(mint));`)
