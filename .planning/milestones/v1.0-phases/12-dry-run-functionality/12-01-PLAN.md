<objective>
Implement the backend dry-run interceptor layer: config flag, schema migration, gate points in broadcaster and jito-seller, trade-store threading, recovery skip, and position manager log-only triggers.

Purpose: When `dryRun: true`, the bot runs the full mainnet pipeline (detection, safety checks, transaction building) but intercepts at two gate points before any signing/broadcasting occurs. Shadow price tracking via PositionManager continues to show what P&L would have been.

Output: All backend files updated, dry-run trades flow through full state machine, all existing + new tests pass.
</objective>
